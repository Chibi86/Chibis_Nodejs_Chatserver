Setting up a client for a Chibi's Node.js Chatserver.

Open up firebug/webconsole and see the output in the console.


Command What does it do? Example Result
/connect Connect to server that is already choosen
/connect [server-address] Coonect to choosen server /connect ws://127.0.0.1:1337
/quit Disconnect to chat-server
/quit [msg] Disconnect to chat-server with message /quit brb reboot computer [nick] leaving (brb reboot)
/nick [new nick] Change user nick (Between 2 to 15 chars) /nick Chibi
/nickcolor [color] Change nick-color (rgb-, hex- or hsl-color) /nickcolor rgb(87, 62, 141) [nick]
/color [color] [message] Textcolor on message (rgb-, hex- or hsl-color) /color #F70033 Red! [nick]: Red!
/msg [nick] [message] Private message in chat-window to choosen user /msg Chibi Secret! [nick]: *Chibi* Secret!
/notice [nick] [message] Message to choosen user in chat-window /notice Chibi Hello! [nick]: -Chibi- Hello!
/slap [nick] Slap choosen user in cyberspace /slap Chibi [nick] slaps Chibi around a bit with a large trout!
/me [message] Write as third person /me study now * [nick] study now
